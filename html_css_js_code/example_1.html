Welcome to HTML!

HTML stands for HyperText Markup Language. It's the standard markup language for documents designed to be displayed in a web browser.

Basic Elements

Here are some basic HTML elements:

  • Headings - Defined with <h1> to <h6>. <h1> is the most important heading.
  • Paragraph - Defined with <p>. This is a paragraph of text.
  • Links - Defined with <a href=\"URL\">. For example, this is a link.
  • Images - Defined with <img src=\"imageURL\" alt=\"description\">. The alt attribute provides alternative text for the image if it cannot be displayed.
